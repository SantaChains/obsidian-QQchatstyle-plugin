# ChatStyle 插件功能演示

## 基础聊天界面

```chat
#title=好友聊天
#theme=linear-gradient(135deg, #667eea 0%, #764ba2 100%)

@小明(pic:明, col:#4a90e2, L)
@小红(pic:红, col:#e91e63, R)

1小明:你好小红！今天天气真不错 ☀️
2小红:是的呢！要不要一起出去走走？
3小明:>小红:是的呢！要不要一起出去走走？
好主意！我们去公园吧 🌳
4小红:太棒了！我带上相机 📷
```

## 自定义颜色聊天

```chat
#title=彩虹聊天室

@用户A(pic:A, col:#ff6b6b, L)
@用户B(pic:B, col:#4ecdc4, R)
@用户C(pic:C, col:#45b7d1, L)

1(#ffeb3b)用户A:这是黄色的消息！
2用户B:我用默认的青色
3(#9c27b0)用户C:紫色看起来很酷
4(#ff5722)用户A:现在换成橙色
5用户B:颜色真丰富啊！
```

## 引用消息示例

```chat
@张三
@李四
@王五

1张三:大家觉得这个方案怎么样？
2李四:我觉得不错，有几个细节需要调整
3王五:>张三:大家觉得这个方案怎么样？
整体思路很好，我赞成李四的意见
4张三:>李四:我觉得不错，有几个细节需要调整
具体是哪些细节？
5李四:主要是时间安排和资源分配
```

## Markdown 格式支持

```chat
@技术大佬(pic:技, col:#2196f3, L)
@新手小白(pic:新, col:#4caf50, R)

1技术大佬:你可以使用**粗体**和*斜体*来强调重点
2新手小白:还可以用`代码块`吗？
3技术大佬:当然可以！还支持[[双向链接]]
4新手小白:太棒了！学到了很多
```

## 简化语法示例

```chat
@老师
@学生

1老师:今天我们学习React hooks
2学生:useState和useEffect有什么区别？
3老师:>2
useState管理状态，useEffect处理副作用
4学生:明白了，谢谢老师！
```
